# Project: [BLOG]

## Overview

동영상, 사진, 글작성 업로드가 가능한 개발 블로그

## Tech Stack

- Language: [TypeScript, javascript]
- Framework: [React]
- Database: [MySQL]

## Project Structure

```
src/
├── components/    # UI 컴포넌트
├── pages/         # 페이지
├── utils/         # 유틸리티 함수
└── styles/        # 스타일
```

## Architecture

- Presentational/Container 패턴으로 기능/화면으로 관심사 분리

## Language Rules

- 모든 결과값과 설명은 반드시 한글로 작성한다.
