# 개인 블로그
https://devnote.blog

## 설명
포스팅 작업이 가능한 개인 블로그 입니다. 

### 주요기능
- 소셜 로그인 
- Tiptap 라이브러리와 extension을 바탕으로 한 포스트 작성 기능 
- 댓글 알림 실시간 확인
